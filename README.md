# 🛒 Store - E-Commerce Web App
This is a modern e-commerce platform built with **Next.js**. It features a clean UI and is designed for a seamless shopping experience.

## 🚀 Key Features
*   **Dynamic Product Listing:** Browse through various categories.
*   **Responsive Design:** Fully optimized for mobile, tablet, and desktop.
*   **Optimized Performance:** Built using Next.js 14 and Geist font for speed and style.
*   **Fast Loading:** Utilizes `next/font` for automatic optimization.

## 🛠️ Getting Started
To run this project locally:

1. Clone the repo:
   ```bash
   git clone https://github.com
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## 🏗️ Tech Stack
* **Framework:** [Next.js](https://nextjs.org/)
* **Styling:** CSS Modules / Tailwind CSS
* **Deployment:** [Vercel](https://vercel.com/)
